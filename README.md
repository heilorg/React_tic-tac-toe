React tic-tac-toe
--------

### 리액트 공식 사이트의 예제를 따라 쳐본 틱택토 프로그램
출처 : (https://ko.reactjs.org/tutorial/tutorial.html)

> 주요기능

* 리액트
* 리액트 돔
* 리액트 이벤트 처리
* 리액트 컴포넌트
* 게임 히스토리 저장, 불러오기


> 사용법

1. 평범한 틱택토와 게임 방식은 동일
2. 오른쪽의 히스토리 버튼을 클릭하면 그 턴의 상황으로 되돌아간다.


> 추후 수정 계획

1. 디자인 추가
2. 더 나은 구조 찾아보기
3. 내가 직접 코드, 구조 짜보기
